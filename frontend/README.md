# LetouMe Frontend

独立 React 前端，使用 `Vite + React + TypeScript + React Router + TanStack Query`。

## 开发

```bash
npm install
cp .env.example .env
npm run dev
```

默认开发地址：

- 前端 `http://localhost:5173`
- 后端 API `http://localhost:8000`

## 命令

```bash
npm run dev
npm run build
npm run test
npm run lint
```
